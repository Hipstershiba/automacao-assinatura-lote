=== INSTRUÇÕES ===

1. Instalar dependências:
   pip install -r requirements.txt

2. Testar (sem build, sem acessar DimensaSign):
   python app.py
   → Ative "🧪 Modo de Teste" nas Configurações Avançadas > Navegador
   → Ou edite config.yaml: test_mode: true

3. Rodar script standalone:
   python src/assinatura.py

4. Build do executável (modo onedir — rápido de abrir):
   python build.py

5. Executável gerado em:
   ./dist/assinatura-lote/assinatura-lote.exe

=== MODO DE TESTE ===

Quando test_mode = true:
- Não abre navegador real
- Gera 15 contratos falsos para processar
- Simula login, token, assinaturas
- Gera relatório Excel normalmente
- Perfeito para validar o app sem o portal DimensaSign

=== ESTRUTURA ===

app.py              → Launcher GUI (ponto de entrada único)
src/assinatura.py   → Módulo de automação
src/dimensa_client.py → Cliente Selenium para DimensaSign
src/dimensa_mock.py → Cliente mock para testes sem portal
src/contrato.py     → Modelo de contrato
src/gerador_relatorio.py → Gerador de relatórios Excel
config.yaml.example → Template de configuração
build.py            → Script de build (PyInstaller, onedir)
