========================================
assinatura-lote — Build & Uso
========================================

📁 Estrutura do executável (--onedir):
  dist/assinatura-lote/
  ├── assinatura-lote.exe   ← executável (5.9 MB)
  ├── config.yaml.example   ← copie para config.yaml e configure
  └── _internal/            ← dependências e módulos

🚀 Para usar:
  1. Copie a pasta dist/assinatura-lote/ para o Windows
  2. Renomeie _internal/src/config.yaml.example → config.yaml
     (ou coloque config.yaml na mesma pasta do .exe)
  3. Execute assinatura-lote.exe

🧪 Modo Mock (testar sem DimensaSign):
  Na GUI, marque "Modo Teste" e "Mock Web"
  Ou no config.yaml:
    test_mode: true
    mock_web: true

🔧 Para rebuildar (precisa de Python):
  python -m venv .venv
  .venv\Scripts\activate
  pip install -r requirements.txt
  python build.py
